指令

发表于 2019-12-17 分类于 计算机组成原理

MIPS 指令集

  • 指令集 一个给定的计算机体系结构所包含的指令集集合
  • 存储程序概念 多种类型的指令和数据均以数字形式存储于存储器中的盖帘，存储程序型计算机即来源于此

MIPS

32 个寄存器

$2^{30}$个存储器字

  • 字 计算机中的基本访问单元 通常时32位为一组

  • 设计原则1 简单源于规整

  • 设计原则2 越小越快

存储器操作数

  • 数据传送指令 再存储器和寄存器之间移动数据的命令
  • 地址 用于在存储器空间中指明某特定数据元素位置的值

常数或立即数操作数

有符号数和无符号数

  • 最高有效位 在MIPS字中最右边的一位
  • 最低有效位 在MIPS字中最左边的一位

原码 反码 补码

偏移表示法 加偏移表示数字 如 1111 表示最大 0000 表示最小

指令的表示

指令的布局形式叫做指令格式

指令的数字形式称为机器语言 这样的指令序列称为机器码

  • 指令格式 二进制数字段组成的指令表示形式
  • 机器语言 在计算机系统中用于交流的二进制表示形式

MIPS 字段

  • 操作码 指令中用来表示操作和格式的字段

R 型(用于寄存器)

op rs rt rd shamt funct
6bit 5bit 5bit 5bit 5bit 6bit
  • op 指令的基本操作，即操作码
  • rs 第一个源操作数寄存器
  • rt 第二个源操作数寄存器
  • rd 用于存放操作结果的目的寄存器
  • shamt 位移量
  • funct 功能码 用于指明op字段中操作的特定形式

设计原则3 优秀的设计需要适宜的折中的方案

I 型(用于立即数)

op rs rt constant or address
6bit 5bit 5bit 16bit

分支和跳转中的寻址

跳转指令 指令会变成下形式

j 10000

2 100000
6bit 26bit

条件分支指令还必须指定两个操作数

bne s0,s1, EXIT

5 16 17 EXIT
6bit 5bit 5bit 16bit

地址计算方法为

程序计数器=寄存器+分支地址

这种寻址方式为PC 相对寻址

范围为$2^{16}$ byte

  • PC 相对寻址 一种寻址方式，它将PC和指令中的常数相加作为寻址结果
