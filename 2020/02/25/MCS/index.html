概述

计算机由五大部分组成:
  • 运算器
  • 控制器
  • 存储器
  • 输入设备
  • 输出设备
总线
  • 数据总线DB
  • 地址总线AB
  • 控制总线CB
性能指标

  • 字长

    计算机一次能处理二进制的位数

    字长越长，数的表示范围也越大，运算精度也越高，功能越强，主存容量越大

  • 主屏

计算机CPU的时钟频率，很大程度决定了计算机运算的速度

  • 主存容量
  • 存储速度

存取时间 : 启动一次存储器操作(读或写)到完成该操作所需要的全部时间

存取周期：存储器进行连续两次独立的存储器操作所需要的最小间隔时间

存储速度的快慢影响大

  • 运算速度

普遍采用单位时间内执行指令的平均条数来衡量，用MIPS作为计量单位，即每秒执行百万条指令

微型计算机的应用

  • 科学计算
  • 信息处理
  • 检测及控制系统与装置
  • 通信
  • 数字多媒体技术
  • 计算机辅助系统

微处理器的发展

微处理器使用一片或少数几片大规模集成电路组成的中央处理器

1 四位微处理器

1970年代初

典型产品 Intel 4004, Inteal4040

字长 4位

特点 指令系统比较简单，运算功能较差，价格低廉

应利 面向家电，计算器和二次仪表

2 八位微处理器

1974-1977

字长 8位

特点 指令系统比较完善，运算速度提高一个数量级，寻址能力有所增强，有中断和DMA,汇编，BASIC,FORTRAN

应用 面向家电，智能仪表，工业控制

3 十六位微处理器

1978-1980

特点 指令系统丰富，采用多级中断，多种寻址方式，段式存储结构，配有功能强大的系统软件

应用 工业控制

4 三十二位微处理器

1980年带初开始，之后十余年迅速发展

5 六十四位处理器

2001 年开始

发展趋势

  • 高性能化
  • 形式多样化
  • 多媒体化
  • 网络化
  • 多核结构
  • 嵌入式应用
  • 智能化

16位微处理器Intel

8086微处理器的构成

内部结构

由两个独立的逻辑单元组成:

  1. 总线接口单元BIU: Bus Interface Unit,完成所有总线操作。
  2. 执行单元EU: Execution Unit, 执行指令。

二者并行工作

8086/8088

EU相同，BIU有差别

  1. 8086: 外部DB 16 位， 指令队列为6个字节
  2. 8088: 外部DB 8 位， 指令队列为4个字节

BIU

BIU 组成

  1. 4个16位段寄存器
  2. 16位指令指针
  3. 6字节指令队列
  4. 总线控制逻辑
  5. 20位地址加法器

BIU功能

  1. 取指令
  2. 形成物理地址
  3. 传送数据，实现CPU与内存,I/O端口间的数据传送

指令预取策略:

当指令队列出现2个以上空字节时(8086)，自动按CS:IP 组成20位物理地址，到存储器去取指令，一次取2个字节

EU

EU 组成:

  1. 16位ALU
  2. 16位标志寄存器
  3. EU控制电路
  4. 8个16位通用寄存器

EU 功能: 完成指令所规定的操作

  1. 从BIU指令队列读取指令
  2. 由EU控制电路译码分析
  3. 在EU种计算操作数的16位偏移地址送BIU,由BIU的$\sum$形成20位物理地址
  4. 将取来的操作数送ALU进行指令运算
  5. 结果送寄存器或送到BIU放到内存
  6. 本次操作状态放到标志寄存器中

结构特点

  1. 由BIU,EU 两个独立单元组成
  2. 取指和执行重叠

8086的编程结构

8086 CPU 有14个16位寄存器

数据寄存器

存放操作数据及中间结果

1) AX 累加器，Accumulator Register

2)BS 基址寄存器， Base Register

3)CX 计数器， Counter

4)DX 数据寄存器 Data Register

可高低字节分别寻址

可存16位地址

有专门功能，如计数器，累加器

指针及变址寄存器

存放逻辑地址的偏移量，是20位物理地址的组成成分

1)SP堆栈指针，Stack Pointer ， 指示在堆栈段中栈顶的位置，用于数据进栈，出栈的位置指向

2） BP 基址指针，Base Pointer ， 指示在堆栈中一个数据区的基址位置，用于访问堆栈段中的某个数据。

3） SI 源变址寄存器，Source Index ， 源串指针，指示在数据段中一个源串操作数的位置

4) DI目的变址寄存器，Destination Index 目的串指针，指示在附加段中一个目的串的操作数的位置

段寄存器

存放当前段的段起始地址

CS 代码段寄存器，Code Segment,存放当前执行程序的段起始地址

SS 堆

Pentium 系列微处理器的基本结构

结构特性：

  1. 超标量流水线，超标量为2
  2. 分立Cache，指令数据各8K
  3. 重新设计的浮点运算部件，8级流水
  4. 动态转移预测
  5. 内部结构32位，外部DB64位，属于32位机
  6. 4中工作方式，实地址，保护虚地址，虚拟8086，系统管理

部件有

  1. 总线接口部件
  2. 预取缓冲存储器
  3. 整数运算部件
  4. 转移目标缓冲气
  5. 浮点运算部件
  6. 代码Cache
  7. 数据Cache
  8. 指令译码部件
  9. 控制部件
  10. 整数及浮点数寄存器组
  11. 控制ROM部件

总线接口部件

总线接口部件包括三总线的控制:

全部总线控制信号，独立的32位地址总线，独立的64位数据总线。
