177. 第N高的薪水

题目

编写一个 SQL 查询，获取 Employee 表中第 n 高的薪水（Salary）。

例如上述 Employee 表，n = 2 时，应返回第二高的薪水 200。如果不存在第 n 高的薪水，那么查询应返回 null。

思路

先排序，取前N个，然后取最小，最后再判断是否存在即可
